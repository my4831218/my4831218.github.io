캡슐화

  • 정보 은닉을 활용한 캡슐화
  • 꼭 필요한 정보와 기능만 외부 접근을 허용
  • 대부분 멤버 변수와 메소드를 감추고 외부 통합 인터페이스만 제공하여 일관된 기능 구현
