    I/O stream

  • 구분

    • 대상 기준 : 입력, 출력
      • System 클래스의 표준 입출력
      • System.out.print(): 출력
      • System.in.read() 입력
      • System.err.print() 에러 출력
    • 자료 종류 : 바이트, 문자
      • 바이트 스트림 : 동영상, 음악, 실행 파일등 자료 R/W 사용
      • 문자 스트림 : 바이트 단위 자료 처리시 문자 깨짐, 인코딩에 맞는 2바이트 이상 처리구현 스트림
    • 기능 : 기반, 보조
      • 기반 스트림 : 대상에 직접 자료 읽고 쓰는 기능
      • 보조 스트림 : 기반 스트림 기능 없이 추가적인 기능 더하는 스트림, 항상 기반 or 다른 보조 스트림 생성자의 매개변수로 포함

바이트 단위 입출력 스트림

  • flush() : 출력 버퍼 비우기, close 메소드가 호출함
  • InputStream
    • 바이트 단위 입력 최상위 추상 클래스
    • int read(). void close() 메소드
  • OutputStream
    • 바이트 단위 출력 최상위 추상 클래스
    • int write(). void flush(), void close() 등 메소드

문자 단위 입출력 스트림

  • Reader
    • 문자단위 입력 최상위 추상 클래스
    • int read(), void close()
  • Writer
    • 문자단위 출력 최상위 추상 클래스
    • int write(), int flush, void close()
